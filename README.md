# groovy-tic-tac-toe
A bit extended version of the classic tic tac toe game (game board up to 10x10 squares!) Played from the command line

Build:
In project directory:
`gradle preparejar`
 
Run:
`java -jar build/libs/groovy-tic-tac-toe.jar`
